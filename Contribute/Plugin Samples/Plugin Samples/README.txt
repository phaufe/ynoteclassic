This folder Contains Samples to make plugins for Ynote Classic.

Ynote Plugins can be created in the following languages -

1.)C# 
2.)VB.NET
3.)Python(IronPython)
4.)Ruby(IronRuby)

This folder has samples for C#, VB.NET , Python and Ruby. You can also create Ynote Plugins with Boo.

Required Assembly(s) for Developing a Plugin

1.)SS.Ynote.Classic.exe
2.)SS.Ynote.Engine.dll
3.)FastColoredTextBox.dll(If Styling/Highlighting Plugin)
4.)WeifenLuo.WinformsUI.Docking(Dock Content Inheritance)

Main Class To Inherit if :

1.)Form Plugin : SS.Ynote.Engine.Framework.Plugins.IFormPlugin
2.)UserControl Plugin : SS.Ynote.Engine.Framework.Plugins.IUserControlPlugin

See Samples.
!Only the samples of C# are written and tested.
There is no guarantee on the working of the others.
